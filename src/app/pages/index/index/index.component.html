Lava Rápido (Back-end)

Projetei uma Api feita totalmente com Laravel 10,utilizei token JWT (JavaScript Web Token) para autenticar os usuários,é uma api simples para o usuário temos duas funções no controller que é para cadastrar e autenticar,e para o Lava Rápido temos 3 funções,para puxar os registros dos horários,para marcar horários,e um especifíco para o administrador do site para confirmar os horários,tudo isso feito com validação tanto do pacote Validator quanto por verificações dos registros que já estão no banco de dados como por exemplo verificar se o horário já está marcado para não escolher outro igual.Escolhi para guardar os dados o banco de dados relacional PostgreSQL.
